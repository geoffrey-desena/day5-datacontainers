# simplemath

`simplemath` is a small educational Python package that demonstrates how to
organize simple mathematical functions into a package structure suitable for
importing, testing, and documenting with Sphinx.

## Features

- Basic arithmetic operations:
  - `simple_add`
  - `simple_sub`
  - `simple_mult`
  - `simple_div`

- Polynomial evaluation:
  - `poly_first`
  - `poly_second`
